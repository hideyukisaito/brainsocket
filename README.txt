------------------- はじめに -------------------

1. nave, node.jsのインストール

node.jsは絶賛開発中のため、頻繁にバージョンアップされています。
バージョン管理のため、まずは nave をインストールします。

1.1 適当な場所にワーキングディレクトリを作って移動します.
ex:
$ mkdir work
$ cd work

1.2 gitから最新のパッケージを取得します.
$ git clone http://github.com/isaacs/nave.git

1.3 展開されたnaveディレクトリに移動し、"nave.sh"を実行します.
    失敗する場合はルート権限で実行してください.
$ cd nave
$ ./nave.sh install latest

1.4 使用するnodeのバージョンを最新版にします.
$ ./nave.sh usemain latest

1.5 インストールが終わったらチェックしてみます。
$ node -v

バージョンが表示されればnodeのインストールは成功です。

------------------------------------------------

2. npm (node package manager)のインストール

nodeにおけるパッケージ管理ツールである
npm( http://npmjs.org/ )をインストールします。

2.1 あらかじめ、ホームディレクトリに
    "node_modules"という名前のフォルダを作っておきます.
$ cd ~
$ mkdir node_modules

2.2 以下のコマンドで一発インストール
$ curl http://npmjs.org/install.sh | sh

失敗する場合は
$ sudo curl http://npmjs.org/install.sh | sh

2.3 インストールが終わったらチェックしてみます。
$ npm -v

バージョンが表示されればnpmのインストールは成功です。

------------------------------------------------

3. ライブラリ群のインストール

今回は以下のライブラリを使用しました。
・socket.io
・express
これらをnpmを使ってインストールします。

$ npm install ライブラリ名
を上記のライブラリ分実行してください。

-*-*-*-*-*-*-*-*-* U S A G E *-*-*-*-*-*-*-*-*-
-*-*-*-*-*-*-*-*-*-*-*-*-*-*-*-*-*-*-*-*-*-*-*-

このアプリケーションを実行するには以下のコマンドを
入力します。
$ cd このファイルがあるディレクトリまでのパス
$ node app.js

ポート番号を指定する場合は以下のようにします.
$ node app.js ポート番号

ホストを指定するには以下のようにします.
この場合、ポート番号は必須になります。
$ node app.js ポート番号 ホスト名

例えば、ローカルサーバーが192.168.1.6で動いている場合、
以下のように起動することでiPhoneからもアクセスできます。
$ node app.js 3000 192.168.1.6
Express server running at http://192.168.1.6:3000 //出力
